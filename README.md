## isualizer
* コンパイル不可
* ソースの一部をroute-editorが参照

## route-editor
### コンパイル・実行
    cd route-editor
    make
    ./launcher     

### 操作方法
| 操作       | 機能           |
|:----------:|:--------------:|
| CTRL + S   | 上書き保存 |
| CTRL + Z   | 元に戻す   |
| マウス左(ドラッグ) | 視点移動      |
| マウス右(ドラッグ) | Waypoint移動  |
| マウス右(ダブルクリック) | Waypoint削除（点上） <br> Waypoint挿入（線上） <br> Waypoint追加(その他) |
